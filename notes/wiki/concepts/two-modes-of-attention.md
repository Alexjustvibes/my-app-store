# Concept: Two Modes of Attention (creativity vs. productivity)

The idea that creative knowledge work runs on two opposed, non-simultaneous states, and that trying to do both at once produces neither.

## The split
| | Creative mode | Productive mode |
|---|---|---|
| Koe's name | open focus / big-picture | deep focus |
| Brain network (his framing) | Default Mode Network (DMN) | Task Positive Network (TPN) |
| Does what | **connects** ideas | brings **clarity** / order |
| Active when | you're *off* an external task (walk, exercise, sun, music, rest) | you're *on* a structured, distraction-free block |
| In writing | idea generation, brainstorming, the "aha" | outlining → organizing → editing |

Source: [[sources/dan-koe-creativity-vs-productivity]] · [[thinkers/dan-koe]]

## The human's working version
Most "productivity" advice optimizes only the productive mode (more focus, more blocks, more discipline) and treats rest as the absence of work. This concept inverts that: **the creative mode is also work — it's the load-bearing half — and it's the one the grind model starves.** The two can't be collapsed into one long "writing session"; they have to be *sequenced* (brainstorm → simmer → organize).

This is the same machine as the human's [[the-outline-system]] / Idea Compiling, described in neuro vocabulary instead of process vocabulary. The interesting question is which description is more *useful* to a reader — and whether the neuro story is even load-bearing (see [[seeds/working-practice-wrong-explanation]]).

## Caveat on the science
The DMN/TPN labels are real but applied loosely here; Koe concedes he hasn't researched it. Treat "two modes" as a *practice heuristic that works*, not a validated neuroscience claim. The 11M-bits-vs-50-bits figure traces to Nørretranders/Csikszentmihalyi popularizations.

## Related voice
- **Marie-Louise Von Franz**: "One of the most wicked, destructive forces, psychologically speaking, is unused creative power." The dark mirror of this concept — starving the creative mode isn't neutral; it's corrosive. (Seed-rich; from [[sources/commonplace-2026-06-09]].)

# Notes Performance — running tracker

One place to track every shipped Note against the writing frameworks **and** actual engagement, so hook / tactic / awareness-level learnings compound instead of scattering across seeds.

**Update rule:** add a row on every ship (pull hook-engine/tactics/awareness/concreteness from the seed's "Framework diagnosis"); fill engagement + recognition when the numbers arrive; promote any pattern that emerges up to "Open experiments."

**The metric that matters:** **recognition** (this-is-me replies, saves, quote-reposts that add the reader's own example) > raw likes. Likes are vanity; recognition is the [[concepts/resonance]] signal and the green light to move a seed toward `newsletter`.

---

## Open experiments / hypotheses being tested
1. **Hook engine: numbers/negativity vs. pure conviction.** Discord Note (number + negativity) vs. Writer Note (conviction-only). — *Data so far: Discord (with number) = 1 sub; Alignment note (conviction, NO number) = 1 sub. Both converted regardless of hook engine → conversion may track resonance-with-the-right-person more than hook tactic. Still need view counts + more n.*
2. **Concreteness premium.** Does a specific number/anchor beat a principle-only Note? (Same two Notes.)
3. **Engagement ≠ mission.** The "bloodline" note overperformed while being the most off-strategy (see [[sources/own-notes-batch-pre-wiki]]). Watch for this trap: a Note can win attention and fail the mission test.
